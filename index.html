XportOne – Privacy Policy

Last updated: October 2025

Melo Estrutural (“we,” “our,” or “us”) respects your privacy. This Privacy Policy explains how the XportOne add-in for Autodesk Revit handles information and data.

1. Data Collection and Use

XportOne does not collect, store, or transmit any personal data from users.

The only data temporarily accessed by the app are:

  • Autodesk User ID (LoginUserId): used solely to verify whether the signed-in Autodesk account has a valid license through Autodesk’s official Entitlement API.
  • Project-related data: such as sheet numbers, sheet names, and export settings, processed locally within Revit to perform the export operation.

No project data, user credentials, or usage analytics are transmitted to Darwin Engenharia or to any third party.

2. Third-Party Services

The only external service contacted by XportOne is Autodesk’s App Store Entitlement Web Service (https://apps.autodesk.com/webservices/checkentitlement) for license validation.

This request is handled directly between Revit and Autodesk’s servers; no information is sent to Melo Estrutural. XportOne does not integrate with analytics networks, advertising services, or any other external SDKs.

3. Data Retention and Deletion

XportOne does not maintain any online databases or cloud storage. A small local cache file may be created on the user’s computer to store recent license-check results for offline validation.

This file remains entirely on the user’s machine and can be safely deleted at any time without affecting Autodesk Revit or the user’s account.

4. User Rights and Consent

Because no personal data is collected or transmitted, there is no need for explicit consent or deletion requests.

However, users may delete the local cache file or uninstall the add-in at any time to remove all locally stored information.
